# zhhth-archive

## 简介

zhhth-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，仅作为静态 HTML 页面的集中存放与版本管理之用。每个页面均可独立访问，适合保存网页快照、演示页面、历史版本等资料。

## 目录结构

```
zhhth-archive/
├── pages/          # 存放所有归档的 HTML 页面文件
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
├── index.html      # 可选：仓库入口页或导航页
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件以描述性名称命名，便于识别内容。子目录可按主题或时间归类。
- `assets/`：存放多个页面可能共用的样式、脚本或媒体资源，避免重复。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，内容可能来自不同来源、不同时期。每个页面保留其原始结构与样式，不进行统一修改或重定向。

如需引用或链接到仓库中的某个页面，请直接使用该文件的原始 GitHub 链接（例如 `https://raw.githubusercontent.com/用户名/zhhth-archive/main/pages/example.html`），或通过 GitHub Pages 部署后访问。

> 注意：归档页面仅供存档与参考，不保证其外部链接、资源或脚本的持续可用性。
